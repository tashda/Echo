# <img src="icon.png" alt="Fuzee Logo" width="48"/> Fuzee

A modern, native SQL client for macOS and iPadOS.

## Features

* **Native Performance:** Built with SwiftUI for a fast and fluid experience.
* **Multi-Database Support:** Connect to multiple databases at once.
* **SQL Editor:** A powerful SQL editor with syntax highlighting and autocompletion.
* **Database Explorer:** Easily explore your database schema, tables, views, and more.
* **Cross-Platform:** Works on both macOS and iPadOS.

## Supported Databases

* PostgreSQL (more coming soon!)

## Future Plans

* Support for more databases, including MySQL, SQLite, and Microsoft SQL Server.
* A more advanced query editor with features like tabs, history, and saved queries.
* Data visualization tools.
* And much more!
